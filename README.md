# 🎓 Advanced Analytics Using Statistics Masterclass

Welcome to the ultimate open-source repository for learning and applying statistics in Data Science, Machine Learning, Business Analytics, and academic placements. 

This repository covers **39 modules** of statistics with 100X depth, featuring:
1. **Bilingual Introductions**: Explanations written in **English**, **Marathi (मराठी)**, and **Hinglish** for intuitive learning.
2. **Theory & Math**: Rigorous formulas, mathematical derivations, and visual learning aids.
3. **Python Implementations**: Clean code blocks using Pandas, NumPy, SciPy, StatsModels, and Plotly.
4. **Interactive Dashboard**: A custom single-page web application featuring live statistical calculators (Central Limit Theorem simulations, T-tests, ANOVA calculators, etc.).

## ⚡ Smart Study Dashboard (Web Application)

This workspace contains a **Smart Study Web App** built locally. It operates completely in-browser with **no external server required**.

### How to use it:
1. Open the repository directory on your computer.
2. Double-click the **`index.html`** file to open it in your web browser.
3. **Core Features**:
   * **Language switcher**: Cycle between English, Marathi, and Hinglish.
   * **CLT Simulator**: Set sample sizes dynamically and watch the sampling distribution converge visually into a normal bell curve in real-time.
   * **Statistical Calculator**: Input custom lists of numbers to calculate the Mean, Variance, Standard Deviation, and run a **One-Sample T-Test** instantly!
   * **Placement Quiz Cards**: Practice statistics MCQs and interview questions.
